Tipos de Conexión a Internet

LAN (Red de Área Local)
Es una red que conecta computadoras dentro de un área limitada, como una casa o una oficina. Generalmente se utiliza para compartir recursos como impresoras, archivos, y acceso a internet.
WAN (Red de Área Amplia)
Es una red que cubre un área geográfica extensa, como una ciudad o incluso países. Se utiliza para conectar varias LANs a través de líneas telefónicas, satélites u otros medios.
MAN (Red de Área Metropolitana)
Es una red que cubre un área más grande que la LAN pero más pequeña que la WAN, como una ciudad o un campus universitario. Suele utilizarse para interconectar múltiples edificios.
Internet
Es una red global que conecta miles de millones de dispositivos en todo el mundo, permitiendo la comunicación e intercambio de datos entre ellos a través de diversas tecnologías.
Redes Inalámbricas (Wi-Fi)
Son redes que no requieren cables físicos para la conexión, sino que utilizan ondas de radio para transmitir datos. Son muy populares en hogares, oficinas y lugares públicos.

Padres del Internet
Timothy «Tim» John Berners-Lee

Es un científico de la computación británico, conocido por ser el padre de la World Wide Web. Estableció la primera comunicación entre un cliente y un servidor usando el protocolo HTTP en diciembre de 1990.

Vinton Cerf

Cerf desarrolló el Protocolo de Control de Transmisión/Protocolo de Internet (TCP/IP), que permite la comunicación entre redes diferentes.

Evolución

Fecha Evento Descripción
1969 ARPANET Se crea ARPANET, la primera red de comunicación digital, precursora del internet.
1983 Protocolo TCP/IP Se adopta el protocolo TCP/IP, permitiendo la expansión de la red a nivel mundial.
1991 World Wide Web Tim Berners-Lee crea la World Wide Web, popularizando el uso de internet para el público.
1998 Lanzamiento de Google Google es lanzado, transformando la búsqueda de información en la web.
2004 Redes Sociales Las redes sociales como Facebook empiezan a crecer, cambiando la forma de interactuar en línea.
